# Исправление ошибки "Неправильное имя домена" в Timeweb

## Проблема:
При попытке добавить домен `www.ashot-zebelyan.ru` появляется ошибка: "Неправильное имя домена"

---

## Возможные причины:

### 1. Домен нужно добавлять через настройки приложения

**Важно:** Домен для App Platform нужно добавлять **через настройки приложения**, а не через общий раздел "Домены и SSL"!

### 2. Формат домена

Возможно, нужно добавить сначала основной домен без `www`, а потом `www` добавится автоматически.

---

## Правильный способ добавления домена:

### Вариант 1: Через настройки приложения (РЕКОМЕНДУЕТСЯ)

1. **Вернитесь в настройки приложения:**
   - Нажмите "< Назад" (Back)
   - Или перейдите: App Platform → ваше приложение → "Настройки" → "Домены"

2. **В разделе "Домены" нажмите "Редактировать"**

3. **Добавьте домен:**
   - Сначала добавьте: `ashot-zebelyan.ru` (без www)
   - Потом добавьте: `www.ashot-zebelyan.ru`

4. **Timeweb покажет DNS записи** для добавления в nic.ru

### Вариант 2: Через общий раздел "Домены и SSL"

Если вы добавляете через общий раздел:

1. **Добавьте сначала основной домен:**
   - Введите: `ashot-zebelyan.ru` (без www)
   - Нажмите "Добавить"

2. **Потом добавьте www:**
   - Нажмите "+ Ещё один домен"
   - Введите: `www.ashot-zebelyan.ru`
   - Нажмите "Добавить"

3. **Привяжите домен к приложению:**
   - В выпадающем списке "Привязать к сервису" выберите ваше приложение `ashot-zebelyan`

---

## Если ошибка сохраняется:

### Проверьте формат домена:

- ✅ Правильно: `ashot-zebelyan.ru`
- ✅ Правильно: `www.ashot-zebelyan.ru`
- ❌ Неправильно: `https://ashot-zebelyan.ru` (не нужно указывать протокол)
- ❌ Неправильно: `ashot-zebelyan.ru/` (не нужно слэш в конце)

### Попробуйте другой способ:

1. **Сначала добавьте основной домен** `ashot-zebelyan.ru` (без www)
2. **Проверьте, что он добавился**
3. **Потом добавьте** `www.ashot-zebelyan.ru`

---

## Рекомендация:

**Лучше всего добавлять домен через настройки приложения:**

1. Перейдите: **App Platform** → **ashot-zebelyan** → **"Настройки"** → **"Домены"**
2. Нажмите **"Редактировать"**
3. Добавьте домены там

Это правильный способ для App Platform приложений!

---

## После успешного добавления:

1. **Timeweb покажет DNS записи** для добавления в nic.ru
2. **Добавьте их в nic.ru**
3. **Подождите 10-30 минут** для распространения DNS и выпуска SSL
4. **Сайт будет доступен** по адресу `https://ashot-zebelyan.ru`
