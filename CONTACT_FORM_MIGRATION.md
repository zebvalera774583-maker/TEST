# Миграция для формы контактов

Для работы формы "Узнать стоимость" необходимо выполнить SQL-миграцию в Supabase.

## Шаги:

1. **Откройте Supabase Dashboard:**
   - Перейдите на https://supabase.com/dashboard
   - Войдите в свой аккаунт
   - Выберите ваш проект

2. **Откройте SQL Editor:**
   - В левом меню нажмите на **"SQL Editor"**

3. **Выполните SQL-запрос:**
   - Скопируйте содержимое файла `add_contact_requests_table.sql`
   - Вставьте в SQL Editor
   - Нажмите кнопку **"Run"** (или нажмите Ctrl+Enter / Cmd+Enter)

4. **Проверьте результат:**
   - Вы должны увидеть сообщение об успешном выполнении
   - Таблица `contact_requests` теперь создана в базе данных

## Что делает миграция:

- Создает таблицу `contact_requests` для хранения заявок
- Настраивает политики безопасности (RLS)
- Разрешает всем пользователям отправлять заявки (INSERT)
- Чтение заявок доступно только через service_role key (для админов)

---

**После выполнения миграции:** Форма "Узнать стоимость" будет полностью работать!
