# Добавление кастомного домена в Timeweb

## Текущая ситуация:

В настройках приложения виден только временный домен:
- `zebvalera774583-maker-test-16b8.twc1.net`

**Нужно добавить кастомный домен:** `ashot-zebelyan.ru`

---

## Пошаговая инструкция:

### Шаг 1: Открыть настройки доменов

1. В панели Timeweb вы уже в разделе **"Настройки"** (Settings)
2. Найдите раздел **"Домены"** (Domains)
3. Рядом с доменами должна быть кнопка **"Редактировать"** (Edit)
4. **Нажмите "Редактировать"**

### Шаг 2: Добавить кастомный домен

1. После нажатия "Редактировать" откроется форма
2. Нажмите **"Добавить домен"** или **"+ Добавить"**
3. Введите: `ashot-zebelyan.ru`
4. Нажмите **"Добавить"** или **"Сохранить"**
5. Повторите для `www.ashot-zebelyan.ru`

### Шаг 3: Получить DNS записи

После добавления домена Timeweb покажет:
- **DNS записи**, которые нужно добавить в nic.ru
- Обычно это **CNAME записи**
- Или **A записи** с IP адресом

**Запишите эти записи!**

### Шаг 4: Добавить DNS записи в nic.ru

1. **Войдите в панель управления доменом в nic.ru**
2. **Перейдите в "Управление DNS-записями"** (или "DNS-master")
3. **Удалите старые записи** (если были для Vercel)
4. **Добавьте новые записи**, которые показал Timeweb:

   **Обычно это CNAME записи:**
   
   Для основного домена:
   - **Тип:** CNAME
   - **Имя:** `@` (или `ashot-zebelyan.ru`)
   - **Значение:** `zebvalera774583-maker-test-16b8.twc1.net` (или то, что показал Timeweb)
   - **TTL:** 3600
   
   Для www:
   - **Тип:** CNAME
   - **Имя:** `www`
   - **Значение:** `zebvalera774583-maker-test-16b8.twc1.net` (или то, что показал Timeweb)
   - **TTL:** 3600

5. **Сохраните изменения**

### Шаг 5: Дождаться

После добавления DNS записей нужно подождать:
- **10-30 минут** для распространения DNS
- **10-30 минут** для автоматического выпуска SSL сертификата

---

## Что будет дальше:

1. **Timeweb автоматически выпустит SSL сертификат** для вашего домена
2. **Статус домена изменится** на "Активен" или "Valid"
3. **Сайт будет доступен** по адресу `https://ashot-zebelyan.ru`

---

## Проверка:

### Через 10-30 минут:

1. **Проверьте статус домена в Timeweb:**
   - Должен быть "Активен" или "Valid"
   - SSL должен быть "Активен"

2. **Откройте сайт:**
   - `https://ashot-zebelyan.ru`
   - Должен открываться без ошибок SSL

---

## Если что-то не работает:

1. **Проверьте DNS записи** - правильно ли они добавлены в nic.ru
2. **Подождите еще 10-20 минут** - DNS может распространяться дольше
3. **Очистите кэш DNS** на компьютере: `ipconfig /flushdns` в командной строке
4. **Проверьте статус SSL** в настройках домена в Timeweb

---

## Важно:

- **Временный домен** `zebvalera774583-maker-test-16b8.twc1.net` будет работать сразу
- **Кастомный домен** `ashot-zebelyan.ru` заработает после настройки DNS и выпуска SSL

---

**Нажмите "Редактировать" в разделе "Домены" и добавьте `ashot-zebelyan.ru`!**
