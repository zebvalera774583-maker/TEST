# Привязка домена к приложению App Platform

## Текущая ситуация:

✅ Домен `ashot-zebelyan.ru` уже добавлен в Timeweb  
❌ Но он не привязан к приложению App Platform  
❌ SSL сертификат не выпущен (показывается "Купить сертификат")

---

## Что нужно сделать:

### Шаг 1: Привязать домен к приложению

1. **Перейдите в настройки приложения:**
   - В левом меню выберите **"App Platform"**
   - Найдите и откройте приложение **"ashot-zebelyan"**
   - Перейдите в **"Настройки"** (Settings)
   - Найдите раздел **"Домены"** (Domains)

2. **Добавьте домен в настройках приложения:**
   - Нажмите **"Редактировать"** рядом с доменами
   - Нажмите **"Добавить домен"** или **"+ Добавить"**
   - Введите: `ashot-zebelyan.ru`
   - Нажмите **"Добавить"** или **"Сохранить"**
   - Повторите для `www.ashot-zebelyan.ru`

3. **Timeweb автоматически:**
   - Привяжет домен к приложению
   - Выпустит SSL сертификат (Let's Encrypt) автоматически
   - Покажет DNS записи для настройки (если нужно)

---

## Важно:

### Два разных места для доменов:

1. **"Домены и SSL"** (общий раздел):
   - Здесь домен уже добавлен ✅
   - Но это общий список доменов
   - Нужно привязать к конкретному приложению

2. **Настройки приложения → "Домены"**:
   - Здесь нужно добавить домен для конкретного приложения
   - После добавления домен автоматически привяжется
   - SSL сертификат выпустится автоматически

---

## После привязки домена:

1. **SSL сертификат выпустится автоматически:**
   - Timeweb использует Let's Encrypt
   - Это займет 10-30 минут
   - Статус изменится с "Купить сертификат" на "Активен"

2. **Проверьте DNS записи:**
   - Если Timeweb покажет DNS записи - добавьте их в nic.ru
   - Если домен уже на NS-серверах Timeweb - DNS уже настроен

3. **Подождите 10-30 минут:**
   - Для распространения DNS (если нужно)
   - Для выпуска SSL сертификата

4. **Проверьте работу сайта:**
   - Откройте `https://ashot-zebelyan.ru`
   - Должен открываться без ошибок SSL

---

## Если домен на NS-серверах Timeweb:

Если домен уже размещен на NS-серверах Timeweb (как видно на скриншоте):
- DNS записи уже настроены автоматически
- Нужно только привязать домен к приложению
- SSL сертификат выпустится автоматически

---

## Пошаговая инструкция:

1. **App Platform** → **ashot-zebelyan** → **"Настройки"** → **"Домены"**
2. Нажмите **"Редактировать"**
3. Добавьте `ashot-zebelyan.ru`
4. Добавьте `www.ashot-zebelyan.ru`
5. Сохраните
6. Подождите 10-30 минут для выпуска SSL
7. Проверьте `https://ashot-zebelyan.ru`

---

**Перейдите в настройки приложения и добавьте домен там - это привяжет его к приложению и автоматически выпустит SSL!**
