# Решение проблемы: сайт работает только с VPN

## Диагностика

Если сайт работает только с VPN, возможные причины:

### 1. Блокировка Vercel/CDN
Vercel использует CDN, который может блокироваться некоторыми провайдерами в России.

**Решение:**
- Попробуйте использовать прямую ссылку Vercel: `https://your-project.vercel.app`
- Или используйте альтернативные DNS серверы (Cloudflare 1.1.1.1, Google 8.8.8.8)

### 2. Блокировка Supabase
Если Supabase блокируется, сайт не сможет загружать данные и фото.

**Проверка:**
- Откройте консоль браузера (F12) → вкладка Network
- Посмотрите, какие запросы к Supabase не проходят
- Запросы к `*.supabase.co` должны возвращать 200 OK

**Решение:**
- Если Supabase блокируется, рассмотрите перенос на другой хостинг/БД
- Или настройте прокси для API запросов

### 3. DNS кэширование
Провайдер может кэшировать старые DNS записи.

**Решение:**
```powershell
# Очистить DNS кэш Windows
ipconfig /flushdns

# Использовать альтернативные DNS:
# Cloudflare: 1.1.1.1, 1.0.0.1
# Google: 8.8.8.8, 8.8.4.4
```

### 4. Блокировка на уровне провайдера
Некоторые провайдеры блокируют определенные IP/домены.

**Решение:**
- Свяжитесь с провайдером
- Используйте VPN (как сейчас)
- Или рассмотрите хостинг в России (Yandex Cloud, Selectel, Timeweb)

## Быстрая проверка

1. **Проверьте, что именно не работает:**
   - Откройте сайт без VPN
   - Откройте консоль браузера (F12)
   - Посмотрите ошибки в Console и Network
   - Какие запросы не проходят?

2. **Проверьте доступность Supabase:**
   ```powershell
   curl https://your-project.supabase.co
   ```

3. **Проверьте DNS:**
   ```powershell
   nslookup ashot-zebelyan.ru
   nslookup www.ashot-zebelyan.ru
   ```

## Рекомендации

Если проблема связана с блокировкой Vercel/Supabase:
- Рассмотрите перенос на российский хостинг (Yandex Cloud, Timeweb)
- Или настройте прокси/зеркало для API запросов
- Или используйте альтернативные сервисы БД/Storage

Если проблема только у вас (не у всех пользователей):
- Проверьте настройки DNS на вашем компьютере
- Попробуйте другой браузер
- Попробуйте другой интернет-провайдер/мобильный интернет
