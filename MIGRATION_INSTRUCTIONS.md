# Инструкция по выполнению SQL-миграции

## Добавление колонки caption в таблицу site_photos

**Важно:** Выполните эту миграцию ОДИН РАЗ, чтобы добавить поддержку подписей к фото.

### Шаги:

1. **Откройте Supabase Dashboard:**
   - Перейдите на https://supabase.com/dashboard
   - Войдите в свой аккаунт
   - Выберите ваш проект

2. **Откройте SQL Editor:**
   - В левом меню нажмите на **"SQL Editor"** (или перейдите по ссылке: https://supabase.com/dashboard/project/_/sql)

3. **Выполните SQL-запрос:**
   - Вставьте следующий SQL-код в редактор:

```sql
ALTER TABLE site_photos ADD COLUMN IF NOT EXISTS caption TEXT;
```

4. **Нажмите кнопку "Run"** (или нажмите Ctrl+Enter / Cmd+Enter)

5. **Проверьте результат:**
   - Вы должны увидеть сообщение об успешном выполнении
   - Колонка `caption` теперь добавлена в таблицу `site_photos`

### Альтернативный способ (если используете файл):

Вы также можете:
- Нажать кнопку "Upload file" в SQL Editor
- Выбрать файл `add_caption_column.sql` из проекта
- Нажать "Run"

---

**После выполнения миграции:** Функция добавления подписей к фото будет полностью работать!
