# Ручная настройка DNS для домена на внешних NS-серверах

## Проблема:
Домен `ashot-zebelyan.ru` использует внешние NS-серверы (nic.ru), поэтому Timeweb не может автоматически настроить DNS записи.

**Нужно вручную добавить A-запись в nic.ru**, которая будет указывать на IP адрес приложения: `147.45.99.106`

---

## Решение: Добавить A-запись в nic.ru

### Шаг 1: Войти в панель управления доменом в nic.ru

1. Откройте [nic.ru](https://nic.ru)
2. Войдите в свой аккаунт
3. Перейдите в управление доменом `ashot-zebelyan.ru`

### Шаг 2: Перейти в управление DNS-записями

1. Найдите раздел **"Управление DNS-записями"** или **"DNS-master"**
2. Откройте его

### Шаг 3: Удалить старую A-запись (если есть)

1. Найдите A-запись для `ashot-zebelyan.ru` (если она есть)
2. Удалите её (или измените значение на новый IP)

### Шаг 4: Добавить новую A-запись для основного домена

1. Нажмите **"Добавить запись"** или **"+ Добавить"**
2. Заполните:
   - **Тип:** `A`
   - **Имя:** `@` (или `ashot-zebelyan.ru`)
   - **Значение:** `147.45.99.106` (IP адрес вашего App Platform приложения)
   - **TTL:** `3600` (или по умолчанию)
3. Нажмите **"Сохранить"**

### Шаг 5: Добавить A-запись для www

1. Нажмите **"Добавить запись"** или **"+ Добавить"**
2. Заполните:
   - **Тип:** `A`
   - **Имя:** `www`
   - **Значение:** `147.45.99.106` (тот же IP адрес)
   - **TTL:** `3600` (или по умолчанию)
3. Нажмите **"Сохранить"**

**Альтернатива для www:** Можно использовать CNAME запись:
   - **Тип:** `CNAME`
   - **Имя:** `www`
   - **Значение:** `ashot-zebelyan.ru`
   - **TTL:** `3600`

---

## После добавления DNS записей:

### 1. Подождите распространения DNS

- **10-30 минут** для распространения DNS изменений
- Иногда может занять до часа

### 2. Проверьте DNS записи

В командной строке выполните:
```bash
nslookup ashot-zebelyan.ru
```

Должен показать IP: `147.45.99.106`

### 3. Дождитесь выпуска SSL сертификата

- После того как DNS записи обновятся, Timeweb автоматически выпустит SSL сертификат
- Это займет **10-30 минут**

### 4. Проверьте работу сайта

- Откройте `https://ashot-zebelyan.ru`
- Должен открываться без ошибок SSL

---

## Итоговые DNS записи в nic.ru:

Должны быть следующие записи:

1. **A запись для основного домена:**
   - Тип: `A`
   - Имя: `@` (или `ashot-zebelyan.ru`)
   - Значение: `147.45.99.106`
   - TTL: `3600`

2. **A запись для www:**
   - Тип: `A`
   - Имя: `www`
   - Значение: `147.45.99.106`
   - TTL: `3600`

   **Или CNAME:**
   - Тип: `CNAME`
   - Имя: `www`
   - Значение: `ashot-zebelyan.ru`
   - TTL: `3600`

---

## Важно:

- **Не удаляйте MX записи** (для почты, если они есть)
- **Не удаляйте TXT записи** (для SPF, DMARC, если они есть)
- **Добавьте только A записи** для основного домена и www
